# ZOP

Source: https://sites.google.com/shiprocket.com/sop-shiprocket/home/zop

ZOP ZOP.in for Shopify Channel Features :- Automatic Order Sync:- Pending orders from Shopify are automatically synced into the Zop.in system. Automatic Status Sync:- The status of orders processed through Zop.in is automatically updated on the Shopify platform. Catalog & Inventory Sync:- Active products on Shopify are fetched into the Zop.in system for inventory management. Auto Refund:- Shopify sellers can set up auto-refunds credited as store credits. Abandon Cart Message Updates : Automated WhatsApp messages can be sent to customers about incomplete purchases, potentially increasing conversion rates.
