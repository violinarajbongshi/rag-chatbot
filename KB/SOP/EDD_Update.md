# EDD Update

Source: https://sites.google.com/shiprocket.com/sop-shiprocket/edd-update

EDD Update Earlier we were getting several EDD on the page which was creating confusion hence, there will be only 2 EDDs that will show from now onwards in ICRM : EDD provided by Shiprocket : Following the creation of a pickup, Shiprocket will offer a timestamp showing the expected delivery date (EDD) (please see attached screenshot below). EDD by Courier : The courier's EDD will also display a timestamp if we receive an updated EDD from them at any moment (please see attached screenshot below). The Updated EDD by Shiprocket option will continue to be the default for EDD escalation reasoning.
