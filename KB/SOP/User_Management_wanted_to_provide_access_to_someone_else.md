Search this site Embedded Files Skip to main content Skip to navigation User Management (Seller wants to provide access to someone else) Scenario – Seller wants to share his/her SR panel access to other sellers. Agent action – Good morning/evening/afternoon Sir/Ma’am, I will surely assist you regarding the SR panel access concern . I would like to inform you that:- You can create, update and disable users using user management setting This option is not applicable in LITE plan, hence seller needs to upgrade the plan from Lite to Basic/Advance/Pro. You can update the modules by editing the users using this screen 14 users can be added excluding Main user You can provide the access to share the Buyer’s Information You can only update the name & module access. Shiprocket does not allow the updation of E-mail Id You can deactivate the user from SR Panel. Path to activate the multiple users :- Login on SR panel>>Go to menu bar>>Click on settings>>Go to User role management>>Click on manage user>>Click on add new user>>Fill required details>>Sellers tools (for which you want to share the access to other users)>>Click on add user. If seller wants to change the tool access :- Login on SR panel>>Go to menu bar>>Click on settings>>Go to User role management>>Click on manage user>>Select the user>>Click on toggle to deactivate the user or click on edit>>Select module>>Click on update.
